from

Converts arrays, Promises, iterables into Observables.

of

Emits provided values sequentially and completes.

defer

Factory that builds a fresh Observable on each subscription.

iif

Conditionally choose one of two Observables at subscribe time.

ajax

Makes an HTTP request and emits the AjaxResponse or error.

throwError

Emits no values, just errors immediately. Use for fail-fast flows.

EMPTY

Emits nothing and completes immediately. Useful for no-op branches.

combineLatest

Emits latest values from both inputs whenever any changes (after both have emitted once).

merge

Merges values from both buttons as they are clicked.

concat

Runs two Observables sequentially, only starting the second after the first completes.

forkJoin

Waits for all Observables to complete and emits last values.

zip

Pairs up values by index from multiple Observables.

race

Emits from the first Observable to emit, unsubscribing from the rest.

partition

Splits a source Observable into two based on a predicate.
